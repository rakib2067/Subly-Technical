# Subly Front End Challenge

## Installation & Usage

### Technologies

- `React JS`
- `TypeScript`
- `react testing library`
- `jest`

### Instalation

- Clone or download the repo
- Open terminal and run `npm install`

### Usage - Run App and Tests

**npm start**

- starts client
- serves client on localhost:3000

**npm run test**

- runs tests
